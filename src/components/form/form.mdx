# Form

## Introduction

`Form` is a simple wrapper around the html5 `button` element, with support for
style variants, quick customisation via `getSprinkles` and an `as` prop that
enables polymorphism, (e.g. if you would like to return an `<a>` element, or a
custom `Link` component instead of the normal `<button>` element.).
