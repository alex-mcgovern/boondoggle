# CheckboxGroup

A checkbox allows a user to select multiple items from a list of individual
items, or to mark one individual item as selected.

It is a thin layer on top of the
[React Aria CheckboxGroup component](https://react-spectrum.adobe.com/react-aria/CheckboxGroup.html).
